# Chopsticks & Chai

A cozy mobile chopsticks game with cafe-themed characters, Firebase accounts, friends, lobbies, saves, and Capacitor iOS support.

## Development

npm install
npm run dev

## Build

npm run build
npm run testflight:prep

## Firebase Smoke Test

npm run firebase:smoke

## iOS CI

GitHub Actions runs a build-only iOS simulator check on macOS runners.
